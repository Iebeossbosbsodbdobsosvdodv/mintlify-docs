# Bem-vindo à Zynx Wallet

A plataforma de pagamentos digitais mais simples e segura para seu negócio.

## O que é Zynx Wallet?

Zynx Wallet é uma plataforma de pagamentos que permite você receber pagamentos via PIX, gerenciar sua carteira digital e automatizar transações financeiras.

## Primeiros Passos

Para começar a usar a API, você precisa:

1. Criar uma conta em [zynxwallet.com.br](https://zynxwallet.com.br)
2. Gerar uma API Key no [Dashboard](https://zynxwallet.com.br/dashboard)
3. Integrar a API em sua aplicação

## Recursos Principais

- **Pagamentos PIX** - Receba pagamentos instantaneamente
- **Saques** - Transfira saldo para sua conta bancária
- **Transferências** - Envie saldo entre usuários
- **Webhooks** - Receba notificações em tempo real
- **API REST** - Integração simples e documentada
