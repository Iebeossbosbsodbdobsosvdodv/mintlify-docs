# Obter Saldo

Obtenha o saldo atual da sua conta.

## Endpoint

```
GET /v1/account/balance
```

## Autenticação

Requer autenticação via API Key ou JWT Token.

## Exemplo de Requisição

### cURL

```bash
curl -X GET https://api.zynxwallet.com.br/v1/account/balance \
  -H "X-API-Key: sua_api_key_aqui"
```

### JavaScript

```javascript
async function getBalance() {
  const response = await fetch(
    'https://api.zynxwallet.com.br/v1/account/balance',
    {
      headers: {
        'X-API-Key': process.env.ZYNX_API_KEY
      }
    }
  );

  const data = await response.json();
  
  if (!data.success) {
    throw new Error(data.message);
  }

  return data.data;
}

// Usar
const balance = await getBalance();
console.log(`Saldo: R$ ${balance.total.toFixed(2)}`);
console.log(`Disponível: R$ ${balance.available.toFixed(2)}`);
console.log(`Pendente: R$ ${balance.pending.toFixed(2)}`);
```

### Python

```python
import requests

def get_balance():
    response = requests.get(
        'https://api.zynxwallet.com.br/v1/account/balance',
        headers={
            'X-API-Key': os.getenv('ZYNX_API_KEY')
        }
    )

    data = response.json()
    
    if not data['success']:
        raise Exception(data['message'])

    return data['data']

# Usar
balance = get_balance()
print(f"Saldo: R$ {balance['total']:.2f}")
print(f"Disponível: R$ {balance['available']:.2f}")
print(f"Pendente: R$ {balance['pending']:.2f}")
```

## Resposta de Sucesso

```json
{
  "success": true,
  "data": {
    "total": 1500.50,
    "available": 1200.50,
    "pending": 300.00,
    "currency": "BRL",
    "last_updated": "2024-02-10T12:30:00Z"
  }
}
```

## Campos da Resposta

| Campo | Tipo | Descrição |
|-------|------|-----------|
| `total` | number | Saldo total da conta |
| `available` | number | Saldo disponível para usar |
| `pending` | number | Saldo em transações pendentes |
| `currency` | string | Moeda (sempre "BRL") |
| `last_updated` | string | Última atualização |

## Entender os Saldos

- **Total**: Soma de `available` + `pending`
- **Available**: Saldo que você pode usar imediatamente
- **Pending**: Saldo em transações que ainda não foram confirmadas
