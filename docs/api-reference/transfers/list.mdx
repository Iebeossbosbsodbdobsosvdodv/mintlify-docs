# Listar Transferências

Liste todas as suas transferências com suporte a filtros, ordenação e paginação.

## Endpoint

```
GET /v1/transfers
```

## Autenticação

Requer autenticação via API Key ou JWT Token.

## Parâmetros de Query

| Parâmetro | Tipo | Padrão | Descrição |
|-----------|------|--------|-----------|
| `page` | number | 1 | Número da página |
| `limit` | number | 20 | Itens por página (máx 100) |
| `type` | string | - | Tipo (sent, received) |
| `sort` | string | -created_at | Ordenação |
| `from_date` | string | - | Data inicial (ISO 8601) |
| `to_date` | string | - | Data final (ISO 8601) |

## Exemplo de Requisição

### cURL

```bash
# Listar transferências enviadas
curl -X GET "https://api.zynxwallet.com.br/v1/transfers?type=sent&limit=50" \
  -H "X-API-Key: sua_api_key_aqui"

# Listar transferências recebidas
curl -X GET "https://api.zynxwallet.com.br/v1/transfers?type=received&limit=50" \
  -H "X-API-Key: sua_api_key_aqui"
```

### JavaScript

```javascript
async function listTransfers(options = {}) {
  const params = new URLSearchParams({
    page: options.page || 1,
    limit: options.limit || 20,
    type: options.type || '',
    sort: options.sort || '-created_at'
  });

  const response = await fetch(
    `https://api.zynxwallet.com.br/v1/transfers?${params}`,
    {
      headers: {
        'X-API-Key': process.env.ZYNX_API_KEY
      }
    }
  );

  const data = await response.json();
  
  if (!data.success) {
    throw new Error(data.message);
  }

  return data.data;
}

// Usar
const transfers = await listTransfers({
  type: 'sent',
  limit: 50
});

console.log(`Total enviado: ${transfers.total}`);
```

### Python

```python
import requests

def list_transfers(page=1, limit=20, transfer_type=None):
    params = {
        'page': page,
        'limit': limit
    }
    
    if transfer_type:
        params['type'] = transfer_type

    response = requests.get(
        'https://api.zynxwallet.com.br/v1/transfers',
        headers={
            'X-API-Key': os.getenv('ZYNX_API_KEY')
        },
        params=params
    )

    data = response.json()
    
    if not data['success']:
        raise Exception(data['message'])

    return data['data']

# Usar
transfers = list_transfers(transfer_type='sent', limit=50)
print(f"Total: {transfers['total']}")
```
