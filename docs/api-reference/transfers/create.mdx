# Criar Transferência

Transfira fundos para outro usuário Zynx Wallet instantaneamente.

## Endpoint

```
POST /v1/transfers/create
```

## Autenticação

Requer autenticação via JWT Token.

## Parâmetros

| Parâmetro | Tipo | Obrigatório | Descrição |
|-----------|------|-------------|-----------|
| `recipient_email` | string | Sim | Email do destinatário |
| `amount` | number | Sim | Valor da transferência em reais |
| `description` | string | Não | Descrição da transferência |

## Exemplo de Requisição

### cURL

```bash
curl -X POST https://api.zynxwallet.com.br/v1/transfers/create \
  -H "Authorization: Bearer seu_jwt_token" \
  -H "Content-Type: application/json" \
  -d '{
    "recipient_email": "destinatario@example.com",
    "amount": 250.00,
    "description": "Pagamento de dívida"
  }'
```

### JavaScript

```javascript
async function createTransfer(recipientEmail, amount, description = '') {
  const response = await fetch(
    'https://api.zynxwallet.com.br/v1/transfers/create',
    {
      method: 'POST',
      headers: {
        'Authorization': `Bearer ${getToken()}`,
        'Content-Type': 'application/json'
      },
      body: JSON.stringify({
        recipient_email: recipientEmail,
        amount,
        description
      })
    }
  );

  const data = await response.json();
  
  if (!data.success) {
    throw new Error(data.message);
  }

  return data.data;
}

// Usar
const transfer = await createTransfer(
  'destinatario@example.com',
  250.00,
  'Pagamento de dívida'
);
console.log('Transferência criada:', transfer.id);
```

### Python

```python
import requests

def create_transfer(recipient_email, amount, description='', token=None):
    response = requests.post(
        'https://api.zynxwallet.com.br/v1/transfers/create',
        headers={
            'Authorization': f'Bearer {token}',
            'Content-Type': 'application/json'
        },
        json={
            'recipient_email': recipient_email,
            'amount': amount,
            'description': description
        }
    )

    data = response.json()
    
    if not data['success']:
        raise Exception(data['message'])

    return data['data']

# Usar
transfer = create_transfer(
    'destinatario@example.com',
    250.00,
    'Pagamento de dívida',
    token='seu_jwt_token'
)
print(f"Transferência criada: {transfer['id']}")
```

## Campos da Resposta

| Campo | Tipo | Descrição |
|-------|------|-----------|
| `id` | string | ID único da transferência |
| `amount` | number | Valor transferido |
| `currency` | string | Moeda (sempre "BRL") |
| `status` | string | Status da transferência |
| `sender` | object | Dados do remetente |
| `recipient` | object | Dados do destinatário |
| `description` | string | Descrição da transferência |
| `created_at` | string | Data de criação |
| `completed_at` | string | Data de conclusão |

## Códigos de Erro

### 400 - Bad Request

```json
{
  "success": false,
  "error": "insufficient_balance",
  "message": "Saldo insuficiente para realizar esta transferência"
}
```

**Causas comuns:**
- Saldo insuficiente
- Destinatário não existe
- Valor inválido

### 404 - Not Found

```json
{
  "success": false,
  "error": "recipient_not_found",
  "message": "Destinatário não encontrado"
}
```

## Limites de Transferência

| Limite | Valor |
|--------|-------|
| Mínimo | R$ 1.00 |
| Máximo | R$ 100.000.00 |
| Diário | R$ 500.000.00 |
