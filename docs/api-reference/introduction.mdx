# Introdução à API

A API Zynx Wallet é uma API RESTful moderna que permite integrar pagamentos PIX, gerenciar carteiras digitais e automatizar suas operações financeiras.

## Características Principais

- **RESTful**: Interface padrão e intuitiva
- **JSON**: Requisições e respostas em JSON
- **Autenticação**: Suporte a API Keys e JWT Tokens
- **Rate Limiting**: Limites generosos para escalabilidade
- **Webhooks**: Notificações em tempo real
- **Documentação**: Exemplos em múltiplas linguagens

## Base URL

```
https://api.zynxwallet.com.br/v1
```

## Versões

A API utiliza versionamento de URL. A versão atual é `v1`.

## Autenticação

Todas as requisições devem incluir autenticação. Veja [Autenticação](/getting-started/authentication) para detalhes.

## Formato de Resposta

Todas as respostas seguem este formato:

### Sucesso (2xx)

```json
{
  "success": true,
  "data": {
    // Dados da resposta
  }
}
```

### Erro (4xx, 5xx)

```json
{
  "success": false,
  "error": "error_code",
  "message": "Descrição do erro"
}
```

## Exemplos de Código

### JavaScript

```javascript
const apiKey = 'sua_api_key_aqui';
const baseUrl = 'https://api.zynxwallet.com.br/v1';

async function createPayment(amount, description) {
  const response = await fetch(`${baseUrl}/payments/create`, {
    method: 'POST',
    headers: {
      'X-API-Key': apiKey,
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      amount,
      description
    })
  });
  
  return await response.json();
}
```

### Python

```python
import requests

api_key = 'sua_api_key_aqui'
base_url = 'https://api.zynxwallet.com.br/v1'

def create_payment(amount, description):
    response = requests.post(
        f'{base_url}/payments/create',
        headers={
            'X-API-Key': api_key,
            'Content-Type': 'application/json'
        },
        json={
            'amount': amount,
            'description': description
        }
    )
    return response.json()
```

### cURL

```bash
curl -X POST https://api.zynxwallet.com.br/v1/payments/create \
  -H "X-API-Key: sua_api_key_aqui" \
  -H "Content-Type: application/json" \
  -d '{
    "amount": 100.00,
    "description": "Pagamento de teste"
  }'
```
