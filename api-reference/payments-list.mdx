# Listar Pagamentos


<Badge method="get">GET</Badge>

Liste todos os pagamentos da sua conta.

## Endpoint

```
GET https://api.zynxwallet.com.br/v1/payments
```

## Parâmetros

| Parâmetro | Tipo | Descrição |
|-----------|------|-----------|
| `limit` | number | Número de resultados (padrão: 10) |
| `offset` | number | Deslocamento para paginação |
| `status` | string | Filtrar por status |

## Exemplo de Requisição

```javascript
const response = await fetch(
  'https://api.zynxwallet.com.br/v1/payments?limit=10&status=completed',
  {
    method: 'GET',
    headers: {
      'X-API-Key': 'sua_api_key_aqui'
    }
  }
);

const data = await response.json();
console.log(data);
```
