# <span style="color: #10B981">POST</span> Transferência Interna

Transfira saldo entre usuários Zynx Wallet.

## Endpoint

```
POST https://api.zynxwallet.com.br/v1/transfers
```

## Parâmetros

| Parâmetro | Tipo | Obrigatório | Descrição |
|-----------|------|-------------|-----------|
| `amount` | number | Sim | Valor em reais |
| `recipient_id` | string | Sim | ID do destinatário |
| `description` | string | Não | Descrição da transferência |

## Exemplo de Requisição

```javascript
const response = await fetch('https://api.zynxwallet.com.br/v1/transfers', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'X-API-Key': 'sua_api_key_aqui'
  },
  body: JSON.stringify({
    amount: 100.00,
    recipient_id: 'user_0987654321',
    description: 'Pagamento de dívida'
  })
});

const data = await response.json();
console.log(data);
```

## Resposta

{
  "success": true,
  "data": {

    "status": "completed",

  }
