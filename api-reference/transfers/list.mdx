# Listar Transferências

Liste todas as transferências da sua conta.

## Endpoint

```
GET https://api.zynxwallet.com.br/v1/transfers
```

## Autenticação

Requer API Key no header:

```
X-API-Key: sua_api_key_aqui
```

## Parâmetros

| Parâmetro | Tipo | Obrigatório | Descrição |
|-----------|------|-------------|-----------|
| `limit` | number | Não | Número de resultados (padrão: 10) |
| `offset` | number | Não | Deslocamento para paginação |

## Exemplo de Requisição

```javascript
const response = await fetch(
  'https://api.zynxwallet.com.br/v1/transfers?limit=10',
  {
    method: 'GET',
    headers: {
      'X-API-Key': 'sua_api_key_aqui'
    }
  }
);

const data = await response.json();
console.log(data);
```
