# <span style="color: #F59E0B">WEBHOOK</span> Webhooks

Receba notificações em tempo real sobre eventos importantes na sua conta.

## O que são Webhooks?

Webhooks são notificações automáticas enviadas pela Zynx Wallet para sua aplicação quando eventos importantes ocorrem.

## Configurar Webhook

1. Acesse a [Dashboard](https://dashboard.zynxwallet.com.br)
2. Vá para "Webhooks"
3. Clique em "Adicionar Webhook"
4. Insira sua URL: `https://seu-dominio.com/webhook`
5. Selecione os eventos que deseja receber
6. Salve

## Eventos Disponíveis

| Evento | Descrição |
|--------|-----------|
| `payment.completed` | Pagamento foi completado |
| `payment.expired` | QR Code expirou |
| `withdrawal.completed` | Saque foi processado |
| `transfer.completed` | Transferência foi concluída |

## Estrutura do Webhook

```json
{
  "id": "evt_1234567890",
  "type": "payment.completed",
  "timestamp": "2024-02-10T15:30:00Z",
  "data": {
    "id": "pay_1234567890",
    "amount": 150.50,
    "status": "completed"
  }
}
```

## Validar Assinatura

Cada webhook inclui um header `X-Zynx-Signature` com uma assinatura HMAC-SHA256:

```javascript
const crypto = require('crypto');

function validateWebhook(payload, signature, secret) {
  const hash = crypto
    .createHmac('sha256', secret)
    .update(JSON.stringify(payload))
    .digest('hex');
  
  return hash === signature;
}
```
