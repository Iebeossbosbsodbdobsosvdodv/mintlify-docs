# Criar Pagamento

Crie uma nova transação PIX na Zynx Wallet.

## Endpoint

```
POST https://api.zynxwallet.com.br/v1/payments
```

## Autenticação

Requer API Key no header:

```
X-API-Key: sua_api_key_aqui
```

## Parâmetros

| Parâmetro | Tipo | Obrigatório | Descrição |
|-----------|------|-------------|-----------|
| `amount` | number | Sim | Valor em reais (ex: 150.50) |
| `description` | string | Sim | Descrição da transação |
| `customer_email` | string | Não | Email do cliente |
| `customer_phone` | string | Não | Telefone do cliente |
| `metadata` | object | Não | Dados customizados |

## Exemplo de Requisição

```javascript
const response = await fetch('https://api.zynxwallet.com.br/v1/payments', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'X-API-Key': 'sua_api_key_aqui'
  },
  body: JSON.stringify({
    amount: 150.50,
    description: 'Compra de produtos',
    customer_email: 'cliente@example.com',
    metadata: {
      order_id: '12345'
    }
  })
});

const data = await response.json();
console.log(data);
```

```python
import requests

response = requests.post(
  'https://api.zynxwallet.com.br/v1/payments',
  headers={
    'X-API-Key': 'sua_api_key_aqui'
  },
  json={
    'amount': 150.50,
    'description': 'Compra de produtos',
    'customer_email': 'cliente@example.com',
    'metadata': {
      'order_id': '12345'
    }
  }
)

print(response.json())
```

```bash
curl -X POST https://api.zynxwallet.com.br/v1/payments \
  -H "Content-Type: application/json" \
  -H "X-API-Key: sua_api_key_aqui" \
  -d '{
    "amount": 150.50,
    "description": "Compra de produtos",
    "customer_email": "cliente@example.com",
    "metadata": {
      "order_id": "12345"
    }
  }'
```

## Resposta

### Sucesso (201)

```json
{
  "success": true,
  "data": {
    "id": "pay_1234567890",
    "amount": 150.50,
    "description": "Compra de produtos",
    "status": "pending",
    "qr_code": "00020126580014br.gov.bcb.pix...",
    "qr_code_url": "https://api.zynxwallet.com.br/qr/pay_1234567890.png",
    "expires_at": "2024-02-11T15:30:00Z",
    "created_at": "2024-02-10T15:30:00Z"
  }
}
```

### Erro (400)

```json
{
  "success": false,
  "error": "invalid_amount",
  "message": "O valor deve ser maior que 0"
}
```

## Campos da Resposta

| Campo | Tipo | Descrição |
|-------|------|-----------|
| `id` | string | ID único do pagamento |
| `amount` | number | Valor em reais |
| `description` | string | Descrição da transação |
| `status` | string | Status: pending, completed, expired, cancelled |
| `qr_code` | string | QR Code em formato string |
| `qr_code_url` | string | URL da imagem do QR Code |
| `expires_at` | string | Data de expiração (ISO 8601) |
| `created_at` | string | Data de criação (ISO 8601) |

## Códigos de Erro

| Código | Descrição |
|--------|-----------|
| `invalid_amount` | Valor inválido |
| `invalid_description` | Descrição inválida |
| `invalid_email` | Email inválido |
| `rate_limit_exceeded` | Limite de requisições atingido |
| `unauthorized` | API Key inválida |
