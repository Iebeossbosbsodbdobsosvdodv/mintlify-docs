# Buscar Pagamento

Obtenha detalhes de um pagamento específico.

## Endpoint

```
GET https://api.zynxwallet.com.br/v1/payments/{id}
```

## Autenticação

Requer API Key no header:

```
X-API-Key: sua_api_key_aqui
```

## Parâmetros

| Parâmetro | Tipo | Obrigatório | Descrição |
|-----------|------|-------------|-----------|
| `id` | string | Sim | ID do pagamento (path parameter) |

## Exemplo de Requisição

```javascript
const paymentId = 'pay_1234567890';

const response = await fetch(
  `https://api.zynxwallet.com.br/v1/payments/${paymentId}`,
  {
    method: 'GET',
    headers: {
      'X-API-Key': 'sua_api_key_aqui'
    }
  }
);

const data = await response.json();
console.log(data);
```

```python
import requests

payment_id = 'pay_1234567890'

response = requests.get(
  f'https://api.zynxwallet.com.br/v1/payments/{payment_id}',
  headers={
    'X-API-Key': 'sua_api_key_aqui'
  }
)

print(response.json())
```

```bash
curl -X GET https://api.zynxwallet.com.br/v1/payments/pay_1234567890 \
  -H "X-API-Key: sua_api_key_aqui"
```

```json
{
  "success": false,
  "error": "payment_not_found",
  "message": "Pagamento não encontrado"
}
```

## Status do Pagamento

| Status | Descrição |
|--------|-----------|
| `pending` | Aguardando pagamento |
| `completed` | Pagamento concluído |
| `expired` | QR Code expirou |
| `cancelled` | Pagamento cancelado |
