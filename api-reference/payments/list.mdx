# Listar Pagamentos

Liste todos os pagamentos da sua conta.

## Endpoint

```
GET https://api.zynxwallet.com.br/v1/payments
```

## Autenticação

Requer API Key no header:

```
X-API-Key: sua_api_key_aqui
```

## Parâmetros

| Parâmetro | Tipo | Obrigatório | Descrição |
|-----------|------|-------------|-----------|
| `limit` | number | Não | Número de resultados (padrão: 10, máx: 100) |
| `offset` | number | Não | Deslocamento para paginação (padrão: 0) |
| `status` | string | Não | Filtrar por status |
| `from_date` | string | Não | Data inicial (ISO 8601) |
| `to_date` | string | Não | Data final (ISO 8601) |

## Exemplo de Requisição

```javascript
const response = await fetch(
  'https://api.zynxwallet.com.br/v1/payments?limit=10&status=completed',
  {
    method: 'GET',
    headers: {
      'X-API-Key': 'sua_api_key_aqui'
    }
  }
);

const data = await response.json();
console.log(data);
```

```python
import requests

response = requests.get(
  'https://api.zynxwallet.com.br/v1/payments',
  headers={
    'X-API-Key': 'sua_api_key_aqui'
  },
  params={
    'limit': 10,
    'status': 'completed'
  }
)

print(response.json())
```

```bash
curl -X GET 'https://api.zynxwallet.com.br/v1/payments?limit=10&status=completed' \
  -H "X-API-Key: sua_api_key_aqui"
```
