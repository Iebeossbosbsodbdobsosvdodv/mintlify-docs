# Listar Pagamentos

Liste todos os seus pagamentos com suporte a filtros, ordenação e paginação.

## Endpoint

```
GET /v1/payments
```

## Autenticação

Requer autenticação via API Key ou JWT Token.

## Parâmetros de Query

| Parâmetro | Tipo | Padrão | Descrição |
|-----------|------|--------|-----------|
| `page` | number | 1 | Número da página |
| `limit` | number | 20 | Itens por página (máx 100) |
| `status` | string | - | Filtrar por status (pending, completed, cancelled) |
| `sort` | string | -created_at | Ordenação (-created_at, created_at, amount, -amount) |
| `from_date` | string | - | Data inicial (ISO 8601) |
| `to_date` | string | - | Data final (ISO 8601) |
| `customer_email` | string | - | Filtrar por email do cliente |

## Exemplo de Requisição

### cURL

```bash
# Listar pagamentos da primeira página
curl -X GET "https://api.zynxwallet.com.br/v1/payments?page=1&limit=20" \
  -H "X-API-Key: sua_api_key_aqui"

# Listar apenas pagamentos completados
curl -X GET "https://api.zynxwallet.com.br/v1/payments?status=completed&limit=50" \
  -H "X-API-Key: sua_api_key_aqui"

# Listar pagamentos de um período
curl -X GET "https://api.zynxwallet.com.br/v1/payments?from_date=2024-01-01&to_date=2024-02-10" \
  -H "X-API-Key: sua_api_key_aqui"
```

### JavaScript

```javascript
async function listPayments(options = {}) {
  const params = new URLSearchParams({
    page: options.page || 1,
    limit: options.limit || 20,
    status: options.status || '',
    sort: options.sort || '-created_at'
  });

  const response = await fetch(
    `https://api.zynxwallet.com.br/v1/payments?${params}`,
    {
      headers: {
        'X-API-Key': process.env.ZYNX_API_KEY
      }
    }
  );

  const data = await response.json();
  
  if (!data.success) {
    throw new Error(data.message);
  }

  return data.data;
}

// Usar
const payments = await listPayments({
  status: 'completed',
  limit: 50,
  page: 1
});

console.log(`Total: ${payments.total}`);
console.log(`Pagamentos:`, payments.items);
```

### Python

```python
import requests

def list_payments(page=1, limit=20, status=None, sort='-created_at'):
    params = {
        'page': page,
        'limit': limit,
        'sort': sort
    }
    
    if status:
        params['status'] = status

    response = requests.get(
        'https://api.zynxwallet.com.br/v1/payments',
        headers={
            'X-API-Key': os.getenv('ZYNX_API_KEY')
        },
        params=params
    )

    data = response.json()
    
    if not data['success']:
        raise Exception(data['message'])

    return data['data']

# Usar
payments = list_payments(status='completed', limit=50)
print(f"Total: {payments['total']}")
for payment in payments['items']:
    print(f"  {payment['id']}: R$ {payment['amount']}")
```

## Campos da Resposta

| Campo | Tipo | Descrição |
|-------|------|-----------|
| `total` | number | Total de pagamentos |
| `page` | number | Página atual |
| `limit` | number | Itens por página |
| `pages` | number | Total de páginas |
| `items` | array | Lista de pagamentos |

## Exemplos de Uso

### Paginar Todos os Pagamentos

```javascript
async function getAllPayments() {
  const allPayments = [];
  let page = 1;
  let hasMore = true;

  while (hasMore) {
    const result = await listPayments({ page, limit: 100 });
    allPayments.push(...result.items);
    
    hasMore = page < result.pages;
    page++;
  }

  return allPayments;
}

const allPayments = await getAllPayments();
console.log(`Total de pagamentos: ${allPayments.length}`);
```

### Filtrar Pagamentos Completados

```javascript
async function getCompletedPayments() {
  const result = await listPayments({
    status: 'completed',
    limit: 100,
    sort: '-created_at'
  });

  return result.items;
}

const completed = await getCompletedPayments();
const totalAmount = completed.reduce((sum, p) => sum + p.amount, 0);
console.log(`Total recebido: R$ ${totalAmount.toFixed(2)}`);
```

### Filtrar por Período

```javascript
async function getPaymentsByPeriod(startDate, endDate) {
  const params = new URLSearchParams({
    from_date: startDate.toISOString(),
    to_date: endDate.toISOString(),
    limit: 100
  });

  const response = await fetch(
    `https://api.zynxwallet.com.br/v1/payments?${params}`,
    {
      headers: {
        'X-API-Key': process.env.ZYNX_API_KEY
      }
    }
  );

  const data = await response.json();
  return data.data.items;
}

// Usar
const startDate = new Date('2024-02-01');
const endDate = new Date('2024-02-10');
const payments = await getPaymentsByPeriod(startDate, endDate);
```
