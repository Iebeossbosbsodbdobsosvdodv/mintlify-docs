# Listar Transferências

Liste todas as transferências da sua conta.

## Endpoint

```
GET https://api.zynxwallet.com.br/v1/transfers
```

## Parâmetros

| Parâmetro | Tipo | Descrição |
|-----------|------|-----------|
| `limit` | number | Número de resultados (padrão: 10) |
| `offset` | number | Deslocamento para paginação |

## Exemplo de Requisição

```javascript
const response = await fetch(
  'https://api.zynxwallet.com.br/v1/transfers?limit=10',
  {
    method: 'GET',
    headers: {
      'X-API-Key': 'sua_api_key_aqui'
    }
  }
);

const data = await response.json();
console.log(data);
```
