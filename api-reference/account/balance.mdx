# Consultar Saldo

Obtenha o saldo da sua conta Zynx Wallet.

## Endpoint

```
GET https://api.zynxwallet.com.br/v1/account/balance
```

## Autenticação

Requer API Key no header:

```
X-API-Key: sua_api_key_aqui
```

## Exemplo de Requisição

```javascript
const response = await fetch(
  'https://api.zynxwallet.com.br/v1/account/balance',
  {
    method: 'GET',
    headers: {
      'X-API-Key': 'sua_api_key_aqui'
    }
  }
);

const data = await response.json();
console.log(data);
```

```python
import requests

response = requests.get(
  'https://api.zynxwallet.com.br/v1/account/balance',
  headers={
    'X-API-Key': 'sua_api_key_aqui'
  }
)

print(response.json())
```

## Resposta

### Sucesso (200)

```json
{
  "success": true,
  "data": {
    "balance": 1500.50,
    "currency": "BRL",
    "last_updated": "2024-02-10T15:30:00Z"
  }
}
```

## Campos da Resposta

| Campo | Tipo | Descrição |
|-------|------|-----------|
| `balance` | number | Saldo em reais |
| `currency` | string | Moeda (sempre BRL) |
| `last_updated` | string | Última atualização (ISO 8601) |
