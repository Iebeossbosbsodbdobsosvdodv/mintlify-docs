# Criar Saque

Solicite um saque para sua conta bancária.

## Endpoint

```
POST https://api.zynxwallet.com.br/v1/withdrawals
```

## Autenticação

Requer API Key no header:

```
X-API-Key: sua_api_key_aqui
```

## Parâmetros

| Parâmetro | Tipo | Obrigatório | Descrição |
|-----------|------|-------------|-----------|
| `amount` | number | Sim | Valor em reais |
| `bank_account_id` | string | Sim | ID da conta bancária |
| `description` | string | Não | Descrição do saque |

## Exemplo de Requisição

```javascript
const response = await fetch('https://api.zynxwallet.com.br/v1/withdrawals', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'X-API-Key': 'sua_api_key_aqui'
  },
  body: JSON.stringify({
    amount: 500.00,
    bank_account_id: 'acc_1234567890',
    description: 'Saque mensal'
  })
});

const data = await response.json();
console.log(data);
```

```python
import requests

response = requests.post(
  'https://api.zynxwallet.com.br/v1/withdrawals',
  headers={
    'X-API-Key': 'sua_api_key_aqui'
  },
  json={
    'amount': 500.00,
    'bank_account_id': 'acc_1234567890',
    'description': 'Saque mensal'
  }
)

print(response.json())
```

## Resposta

### Sucesso (201)

```json
{
  "success": true,
  "data": {
    "id": "wd_1234567890",
    "amount": 500.00,
    "status": "pending",
    "bank_account_id": "acc_1234567890",
    "description": "Saque mensal",
    "created_at": "2024-02-10T15:30:00Z",
    "estimated_arrival": "2024-02-12T00:00:00Z"
  }
}
```

## Status do Saque

| Status | Descrição |
|--------|-----------|
| `pending` | Aguardando processamento |
| `processing` | Sendo processado |
| `completed` | Concluído |
| `failed` | Falhou |
| `cancelled` | Cancelado |
