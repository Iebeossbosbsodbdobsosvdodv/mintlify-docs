# Obter Saque

Obtenha os detalhes completos de um saque específico.

## Endpoint

```
GET /v1/withdrawals/{withdrawal_id}
```

## Autenticação

Requer autenticação via API Key ou JWT Token.

## Parâmetros

| Parâmetro | Tipo | Localização | Descrição |
|-----------|------|------------|-----------|
| `withdrawal_id` | string | URL | ID do saque |

## Exemplo de Requisição

### cURL

```bash
curl -X GET https://api.zynxwallet.com.br/v1/withdrawals/wd_abc123def456 \
  -H "X-API-Key: sua_api_key_aqui"
```

### JavaScript

```javascript
async function getWithdrawal(withdrawalId) {
  const response = await fetch(
    `https://api.zynxwallet.com.br/v1/withdrawals/${withdrawalId}`,
    {
      headers: {
        'X-API-Key': process.env.ZYNX_API_KEY
      }
    }
  );

  const data = await response.json();
  
  if (!data.success) {
    throw new Error(data.message);
  }

  return data.data;
}

// Usar
const withdrawal = await getWithdrawal('wd_abc123def456');
console.log('Status:', withdrawal.status);
console.log('Valor:', withdrawal.amount);
```

### Python

```python
import requests

def get_withdrawal(withdrawal_id):
    response = requests.get(
        f'https://api.zynxwallet.com.br/v1/withdrawals/{withdrawal_id}',
        headers={
            'X-API-Key': os.getenv('ZYNX_API_KEY')
        }
    )

    data = response.json()
    
    if not data['success']:
        raise Exception(data['message'])

    return data['data']

# Usar
withdrawal = get_withdrawal('wd_abc123def456')
print(f"Status: {withdrawal['status']}")
print(f"Valor: R$ {withdrawal['amount']}")
```

## Códigos de Erro

### 404 - Not Found

```json
{
  "success": false,
  "error": "withdrawal_not_found",
  "message": "Saque não encontrado"
}
```

**Solução:** Verifique se o ID do saque está correto.
