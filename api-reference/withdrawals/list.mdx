# Listar Saques

Liste todos os seus saques com suporte a filtros, ordenação e paginação.

## Endpoint

```
GET /v1/withdrawals
```

## Autenticação

Requer autenticação via API Key ou JWT Token.

## Parâmetros de Query

| Parâmetro | Tipo | Padrão | Descrição |
|-----------|------|--------|-----------|
| `page` | number | 1 | Número da página |
| `limit` | number | 20 | Itens por página (máx 100) |
| `status` | string | - | Filtrar por status |
| `sort` | string | -created_at | Ordenação |
| `from_date` | string | - | Data inicial (ISO 8601) |
| `to_date` | string | - | Data final (ISO 8601) |

## Exemplo de Requisição

### cURL

```bash
# Listar saques da primeira página
curl -X GET "https://api.zynxwallet.com.br/v1/withdrawals?page=1&limit=20" \
  -H "X-API-Key: sua_api_key_aqui"

# Listar apenas saques completados
curl -X GET "https://api.zynxwallet.com.br/v1/withdrawals?status=completed&limit=50" \
  -H "X-API-Key: sua_api_key_aqui"
```

### JavaScript

```javascript
async function listWithdrawals(options = {}) {
  const params = new URLSearchParams({
    page: options.page || 1,
    limit: options.limit || 20,
    status: options.status || '',
    sort: options.sort || '-created_at'
  });

  const response = await fetch(
    `https://api.zynxwallet.com.br/v1/withdrawals?${params}`,
    {
      headers: {
        'X-API-Key': process.env.ZYNX_API_KEY
      }
    }
  );

  const data = await response.json();
  
  if (!data.success) {
    throw new Error(data.message);
  }

  return data.data;
}

// Usar
const withdrawals = await listWithdrawals({
  status: 'completed',
  limit: 50
});

console.log(`Total: ${withdrawals.total}`);
```

### Python

```python
import requests

def list_withdrawals(page=1, limit=20, status=None):
    params = {
        'page': page,
        'limit': limit
    }
    
    if status:
        params['status'] = status

    response = requests.get(
        'https://api.zynxwallet.com.br/v1/withdrawals',
        headers={
            'X-API-Key': os.getenv('ZYNX_API_KEY')
        },
        params=params
    )

    data = response.json()
    
    if not data['success']:
        raise Exception(data['message'])

    return data['data']

# Usar
withdrawals = list_withdrawals(status='completed', limit=50)
print(f"Total: {withdrawals['total']}")
```
