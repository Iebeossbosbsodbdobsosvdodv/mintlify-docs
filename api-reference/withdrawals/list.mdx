# Listar Saques

Liste todos os saques da sua conta.

## Endpoint

```
GET https://api.zynxwallet.com.br/v1/withdrawals
```

## Autenticação

Requer API Key no header:

```
X-API-Key: sua_api_key_aqui
```

## Parâmetros

| Parâmetro | Tipo | Obrigatório | Descrição |
|-----------|------|-------------|-----------|
| `limit` | number | Não | Número de resultados (padrão: 10) |
| `offset` | number | Não | Deslocamento para paginação |
| `status` | string | Não | Filtrar por status |

## Exemplo de Requisição

```javascript
const response = await fetch(
  'https://api.zynxwallet.com.br/v1/withdrawals?limit=10',
  {
    method: 'GET',
    headers: {
      'X-API-Key': 'sua_api_key_aqui'
    }
  }
);

const data = await response.json();
console.log(data);
```
