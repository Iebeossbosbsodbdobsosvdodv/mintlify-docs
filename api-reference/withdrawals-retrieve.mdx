# <span style="color: #3B82F6">GET</span> Buscar Saque

Obtenha detalhes de um saque específico.

## Endpoint

```
GET https://api.zynxwallet.com.br/v1/withdrawals/{id}
```

## Exemplo de Requisição

```javascript
const withdrawalId = 'wd_1234567890';

const response = await fetch(
  `https://api.zynxwallet.com.br/v1/withdrawals/${withdrawalId}`,
  {
    method: 'GET',
    headers: {
      'X-API-Key': 'sua_api_key_aqui'
    }
  }
);

const data = await response.json();
console.log(data);
```
