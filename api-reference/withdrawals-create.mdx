# <span style="color: #10B981">POST</span> Criar Saque

Solicite um saque para sua conta bancária.

## Endpoint

```
POST https://api.zynxwallet.com.br/v1/withdrawals
```

## Parâmetros

| Parâmetro | Tipo | Obrigatório | Descrição |
|-----------|------|-------------|-----------|
| `amount` | number | Sim | Valor em reais |
| `bank_account_id` | string | Sim | ID da conta bancária |
| `description` | string | Não | Descrição do saque |

## Exemplo de Requisição

```javascript
const response = await fetch('https://api.zynxwallet.com.br/v1/withdrawals', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'X-API-Key': 'sua_api_key_aqui'
  },
  body: JSON.stringify({
    amount: 500.00,
    bank_account_id: 'acc_1234567890',
    description: 'Saque mensal'
  })
});

const data = await response.json();
console.log(data);
```
