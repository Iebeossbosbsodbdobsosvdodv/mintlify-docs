## Consultar Saldo e Estatísticas

Retorna o saldo disponível, valores pendentes e estatísticas de uso da sua conta Zynx.

### Requisição

<RequestExample>

```bash cURL
curl --request GET \
  --url https://api.zynxwallet.com.br/api/v1/user/balance \
  --header 'X-API-Key: zynx_sua_chave'
```

</RequestExample>

<ResponseExample>

```json Exemplo de Resposta
{
  "success": true,
  "data": {
    "balance": {
      "total": 1250050,
      "pendingPayments": 45000,
      "totalWithPending": 1295050
    },
    "statistics": {
      "totalReceived": 5000000,
      "completedPayments": 342
    }
  }
}
```

</ResponseExample>

<Info>
  Os valores de saldo são retornados em **centavos** (ex: 1250050 = R$ 12.500,50).
</Info>

## Categorias de Usuário

A Zynx Wallet possui duas categorias principais de usuários que influenciam as taxas e limites:

| Categoria | Descrição | Taxa PIX |
| :--- | :--- | :--- |
| **WHITE** | Categoria padrão para novos usuários. | R$ 0.50 |
| **BLACK** | Categoria para usuários de alto volume. | Sob consulta |

<Tip>
  Para migrar sua conta para a categoria **BLACK**, entre em contato com nosso suporte comercial.
</Tip>
