# <span style="color: #3B82F6">GET</span> Buscar Pagamento

Obtenha detalhes de um pagamento específico.

## Endpoint

```
GET https://api.zynxwallet.com.br/v1/payments/{id}
```

## Exemplo de Requisição

```javascript
const paymentId = 'pay_1234567890';

const response = await fetch(
  `https://api.zynxwallet.com.br/v1/payments/${paymentId}`,
  {
    method: 'GET',
    headers: {
      'X-API-Key': 'sua_api_key_aqui'
    }
  }
);

const data = await response.json();
console.log(data);
```
