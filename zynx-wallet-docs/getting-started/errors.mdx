# Tratamento de Erros

A API Zynx Wallet retorna erros estruturados para facilitar o tratamento.

## Formato de Erro

Todos os erros seguem este formato:

```json
{
  "success": false,
  "error": "error_code",
  "message": "Descrição do erro em português"
}
```

## Códigos de Erro HTTP

| Código | Significado | Solução |
|--------|------------|---------|
| 400 | Bad Request | Verifique os parâmetros da requisição |
| 401 | Unauthorized | Verifique sua API Key ou Token |
| 403 | Forbidden | Você não tem permissão para acessar |
| 404 | Not Found | O recurso não existe |
| 429 | Too Many Requests | Você excedeu o rate limit |
| 500 | Server Error | Erro no servidor, tente novamente |
| 503 | Service Unavailable | Serviço temporariamente indisponível |

## Códigos de Erro Específicos

### `invalid_request`

```json
{
  "success": false,
  "error": "invalid_request",
  "message": "O parâmetro 'amount' é obrigatório"
}
```

**Solução:** Verifique se todos os parâmetros obrigatórios estão presentes e no formato correto.

### `invalid_api_key`

```json
{
  "success": false,
  "error": "invalid_api_key",
  "message": "API Key inválida ou expirada"
}
```

**Solução:** Verifique sua API Key e gere uma nova se necessário.

### `insufficient_balance`

```json
{
  "success": false,
  "error": "insufficient_balance",
  "message": "Saldo insuficiente para realizar esta operação"
}
```

**Solução:** Adicione fundos à sua conta.

### `payment_already_paid`

```json
{
  "success": false,
  "error": "payment_already_paid",
  "message": "Este pagamento já foi pago"
}
```

**Solução:** Não tente processar o mesmo pagamento duas vezes.

### `rate_limit_exceeded`

```json
{
  "success": false,
  "error": "rate_limit_exceeded",
  "message": "Você excedeu o limite de requisições",
  "retry_after": 60
}
```

**Solução:** Aguarde o tempo indicado em `retry_after` antes de fazer nova requisição.

## Tratamento de Erros em Código

### JavaScript

```javascript
async function createPayment(amount, description) {
  try {
    const response = await fetch('https://api.zynxwallet.com.br/v1/payments/create', {
      method: 'POST',
      headers: {
        'Authorization': 'Bearer seu_token',
        'Content-Type': 'application/json'
      },
      body: JSON.stringify({ amount, description })
    });
    
    const data = await response.json();
    
    if (!data.success) {
      // Tratar erro específico
      switch (data.error) {
        case 'insufficient_balance':
          console.error('Saldo insuficiente');
          break;
        case 'invalid_request':
          console.error('Requisição inválida:', data.message);
          break;
        case 'rate_limit_exceeded':
          console.error('Rate limit excedido. Aguarde', data.retry_after, 'segundos');
          break;
        default:
          console.error('Erro desconhecido:', data.message);
      }
      return null;
    }
    
    return data.data;
  } catch (error) {
    console.error('Erro de rede:', error);
    return null;
  }
}
```

### Python

```python
import requests
import time

def create_payment(amount, description, token):
    try:
        response = requests.post(
            'https://api.zynxwallet.com.br/v1/payments/create',
            headers={
                'Authorization': f'Bearer {token}',
                'Content-Type': 'application/json'
            },
            json={'amount': amount, 'description': description}
        )
        
        data = response.json()
        
        if not data.get('success'):
            error = data.get('error')
            
            if error == 'insufficient_balance':
                print('Saldo insuficiente')
            elif error == 'invalid_request':
                print(f'Requisição inválida: {data.get("message")}')
            elif error == 'rate_limit_exceeded':
                retry_after = data.get('retry_after', 60)
                print(f'Rate limit excedido. Aguarde {retry_after}s')
                time.sleep(retry_after)
                return create_payment(amount, description, token)
            else:
                print(f'Erro desconhecido: {data.get("message")}')
            
            return None
        
        return data.get('data')
    
    except requests.RequestException as e:
        print(f'Erro de rede: {e}')
        return None
```

## Implementar Retry Automático

### JavaScript com Retry

```javascript
async function makeRequestWithRetry(url, options, maxRetries = 3) {
  for (let attempt = 1; attempt <= maxRetries; attempt++) {
    try {
      const response = await fetch(url, options);
      const data = await response.json();
      
      if (data.success) {
        return data;
      }
      
      // Não fazer retry para erros de validação
      if (response.status === 400) {
        throw new Error(data.message);
      }
      
      // Fazer retry para erros de servidor
      if (response.status >= 500) {
        if (attempt < maxRetries) {
          const delay = Math.pow(2, attempt - 1) * 1000;
          console.log(`Tentativa ${attempt} falhou. Aguardando ${delay}ms...`);
          await new Promise(resolve => setTimeout(resolve, delay));
          continue;
        }
      }
      
      throw new Error(data.message);
    } catch (error) {
      if (attempt === maxRetries) {
        throw error;
      }
      
      const delay = Math.pow(2, attempt - 1) * 1000;
      console.log(`Erro: ${error.message}. Tentando novamente em ${delay}ms...`);
      await new Promise(resolve => setTimeout(resolve, delay));
    }
  }
}
```

    console.error(JSON.stringify(logEntry));
