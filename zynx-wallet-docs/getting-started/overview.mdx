# Começando com Zynx Wallet

Escolha por onde deseja começar sua jornada com a Zynx Wallet.

## 3 Passos Simples

### 1️⃣ Crie sua conta

Acesse [zynxwallet.com.br](https://zynxwallet.com.br) e crie sua conta gratuitamente em menos de 2 minutos.

**Passo a passo:**
- Clique em "Criar Conta"
- Insira seu email
- Crie uma senha forte
- Confirme seu email
- Pronto! Sua conta está criada

### 2️⃣ Obtenha sua API Key

Acesse a [Dashboard](https://dashboard.zynxwallet.com.br) para configurar suas credenciais de API.

**Como gerar uma API Key:**
1. Faça login em sua conta
2. Vá para "Configurações" > "API Keys"
3. Clique em "Gerar Nova Chave"
4. Dê um nome para sua chave (ex: "Integração Loja")
5. Copie e guarde em local seguro
6. **Nunca compartilhe sua API Key!**

### 3️⃣ Integre e automatize

Use nossa [API](/api-reference/payments/create) para automatizar seus ganhos e gerenciar sua carteira.

## Base URL

Todas as requisições devem ser feitas para:

```
https://api.zynxwallet.com.br
```
