# Testando Webhooks

Aprenda como testar seus webhooks em um ambiente de desenvolvimento.

## Ferramentas Úteis

### 1. Webhook.site

[Webhook.site](https://webhook.site) é uma ferramenta gratuita para testar webhooks:

1. Acesse https://webhook.site
2. Copie a URL gerada
3. Configure essa URL como seu webhook na [Dashboard](https://dashboard.zynxwallet.com.br)
4. Faça uma ação que dispare o webhook (ex: criar pagamento)
5. Veja a requisição recebida em tempo real

### 2. RequestBin

[RequestBin](https://requestbin.com) oferece funcionalidade similar:

1. Acesse https://requestbin.com
2. Clique em "Create a RequestBin"
3. Copie a URL
4. Configure como seu webhook
5. Veja os webhooks recebidos

## Testar Localmente

### Com ngrok

Use [ngrok](https://ngrok.com) para expor seu servidor local:

```bash
# Instalar ngrok
npm install -g ngrok

# Expor porta 3000
ngrok http 3000

# Você receberá uma URL como:
# https://abc123.ngrok.io
```

Então configure seu webhook como `https://abc123.ngrok.io/webhook`.

### Com localhost.run

Alternativa ao ngrok:

```bash
# Expor porta 3000
ssh -R 80:localhost:3000 localhost.run

# Você receberá uma URL como:
# https://abc123.localhost.run
```

## Implementar Endpoint de Teste

### Express.js

```javascript
const express = require('express');
const app = express();

app.use(express.json());

app.post('/webhook', (req, res) => {
  const event = req.body;
  
  console.log('Webhook recebido:');
  console.log('Tipo:', event.type);
  console.log('Dados:', JSON.stringify(event.data, null, 2));
  
  // Validar assinatura
  const signature = req.headers['x-zynx-signature'];
  console.log('Assinatura:', signature);
  
  res.json({ success: true });
});

app.listen(3000, () => {
  console.log('Servidor rodando em http://localhost:3000');
});
```

### Python (Flask)

```python
from flask import Flask, request, jsonify

app = Flask(__name__)

@app.route('/webhook', methods=['POST'])
def webhook():
    event = request.json
    
    print('Webhook recebido:')
    print(f'Tipo: {event["type"]}')
    print(f'Dados: {event["data"]}')
    
    signature = request.headers.get('X-Zynx-Signature')
    print(f'Assinatura: {signature}')
    
    return jsonify({'success': True})

if __name__ == '__main__':
    app.run(debug=True, port=3000)
```

## Simular Webhooks

### Usando cURL

```bash
curl -X POST http://localhost:3000/webhook \
  -H "Content-Type: application/json" \
  -H "X-Zynx-Signature: test_signature" \
  -d '{
    "id": "evt_test123",
    "type": "payment.completed",
    "timestamp": "2024-02-10T12:30:00Z",
    "data": {
      "id": "pay_test123",
      "amount": 150.50,
      "status": "completed"
    }
  }'
```

### Usando Node.js

```javascript
const http = require('http');

function sendWebhook(payload) {
  const data = JSON.stringify(payload);
  
  const options = {
    hostname: 'localhost',
    port: 3000,
    path: '/webhook',
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
      'Content-Length': data.length,
      'X-Zynx-Signature': 'test_signature'
    }
  };
  
  const req = http.request(options, (res) => {
    console.log(`Status: ${res.statusCode}`);
    res.on('data', (chunk) => {
      console.log(`Resposta: ${chunk}`);
    });
  });
  
  req.write(data);
  req.end();
}

// Simular webhook
sendWebhook({
  id: 'evt_test123',
  type: 'payment.completed',
  timestamp: new Date().toISOString(),
  data: {
    id: 'pay_test123',
    amount: 150.50,
    status: 'completed'
  }
});
```

## Validar Assinatura em Testes

```javascript
const crypto = require('crypto');

function validateSignature(payload, signature, secret) {
  const hash = crypto
    .createHmac('sha256', secret)
    .update(JSON.stringify(payload))
    .digest('hex');
  
  return hash === signature;
}

// Testar
const payload = {
  id: 'evt_test123',
  type: 'payment.completed',
  data: { id: 'pay_test123' }
};

const secret = 'seu_webhook_secret';
const signature = crypto
  .createHmac('sha256', secret)
  .update(JSON.stringify(payload))
  .digest('hex');

console.log('Válido:', validateSignature(payload, signature, secret)); // true
```

## Checklist de Teste

- [ ] Endpoint responde com status 200
- [ ] Assinatura é validada corretamente
- [ ] Evento é processado corretamente
- [ ] Banco de dados é atualizado
- [ ] Email é enviado (se aplicável)
- [ ] Idempotência funciona (mesmo evento não duplica)
- [ ] Retry automático funciona (simular timeout)
- [ ] Erros são registrados em log
