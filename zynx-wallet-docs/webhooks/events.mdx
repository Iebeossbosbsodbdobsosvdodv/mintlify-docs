# Eventos de Webhook

Referência completa de todos os eventos que podem ser enviados via webhook.

## Pagamentos

### payment.created

Enviado quando um novo pagamento é criado.

```json
{
  "id": "evt_abc123",
  "type": "payment.created",
  "timestamp": "2024-02-10T12:30:00Z",
  "data": {
    "id": "pay_abc123def456",
    "amount": 150.50,
    "currency": "BRL",
    "description": "Compra de produto",
    "status": "pending",
    "customer_email": "cliente@example.com",
    "created_at": "2024-02-10T12:30:00Z"
  }
}
```

### payment.completed

Enviado quando um pagamento é confirmado.

```json
{
  "id": "evt_abc124",

  "data": {

    "currency": "BRL",
    "description": "Compra de produto",
    "status": "completed",

    "payer_name": "João Silva",
```

## Tratamento de Eventos

### Exemplo: Processar Pagamento Completado

```javascript
app.post('/webhook', async (req, res) => {
  const event = req.body;
  
  if (event.type === 'payment.completed') {
    const payment = event.data;
    
    // Atualizar banco de dados
    await db.payments.update(payment.id, {
      status: 'completed',
      paid_at: payment.paid_at
    });
    
    // Enviar email de confirmação
    await sendEmail(payment.customer_email, {
      subject: 'Pagamento Confirmado',
      body: `Seu pagamento de R$ ${payment.amount} foi confirmado!`
    });
    
    // Processar pedido
    await processOrder(payment.id);
  }
  
  res.json({ success: true });
});
```
