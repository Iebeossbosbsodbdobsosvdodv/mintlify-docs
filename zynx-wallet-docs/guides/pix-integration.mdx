# Integração PIX

Aprenda como integrar pagamentos PIX na sua aplicação usando a API Zynx Wallet.

## O que é PIX?

PIX é um sistema de pagamento instantâneo brasileiro que permite transferências de dinheiro 24/7, inclusive em finais de semana e feriados. Com a Zynx Wallet, você pode aceitar pagamentos PIX facilmente.

## Fluxo de Pagamento PIX

```
┌─────────────┐
│   Cliente   │
└──────┬──────┘
       │
       ├─ 1. Vê QR Code
       │
       ├─ 2. Escaneia com app bancário
       │
       ├─ 3. Confirma pagamento
       │
       ▼
┌──────────────────┐
│  Zynx Wallet     │
│  (Seu servidor)  │
└────────┬─────────┘
         │
         ├─ 4. Recebe PIX
         │
         ├─ 5. Envia webhook
         │
         ▼
┌──────────────────┐
│ Sua Aplicação    │
│ (Seu servidor)   │
└──────────────────┘
```

## Passo 1: Criar Pagamento

Crie um pagamento PIX usando a API:

```javascript
async function createPixPayment(amount, description) {
  const response = await fetch('https://api.zynxwallet.com.br/v1/payments/create', {
    method: 'POST',
    headers: {
      'X-API-Key': process.env.ZYNX_API_KEY,
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      amount,
      description,
      customer_email: 'cliente@example.com'
    })
  });

  const data = await response.json();
  
  if (!data.success) {
    throw new Error(data.message);
  }

  return data.data;
}

// Usar
const payment = await createPixPayment(150.50, 'Compra de produto');
console.log('ID:', payment.id);
console.log('QR Code:', payment.qr_code);
console.log('URL da imagem:', payment.qr_code_url);
```

### React

```jsx
import { useState, useEffect } from 'react';

export function PixPayment({ paymentId }) {
  const [payment, setPayment] = useState(null);
  const [loading, setLoading] = useState(true);

  useEffect(() => {
    async function loadPayment() {
      try {
        const response = await fetch(
          `https://api.zynxwallet.com.br/v1/payments/${paymentId}`,
          {
            headers: {
              'X-API-Key': process.env.REACT_APP_API_KEY
            }
          }
        );
        
        const data = await response.json();
        setPayment(data.data);
      } finally {
        setLoading(false);
      }
    }

    loadPayment();
  }, [paymentId]);

  if (loading) return <div>Carregando...</div>;
  if (!payment) return <div>Erro ao carregar pagamento</div>;

  return (
    <div className="pix-payment">
      <h2>Escaneie o QR Code para pagar</h2>
      <img 
        src={payment.qr_code_url} 
        alt="QR Code PIX"
        className="qr-code"
      />
      <p className="amount">R$ {payment.amount.toFixed(2)}</p>
      <p className="description">{payment.description}</p>
      <p className="status">Status: {payment.status}</p>
    </div>
  );
}
```

## Passo 3: Monitorar Pagamento

### Opção A: Polling (Não Recomendado)

```javascript
async function waitForPayment(paymentId, maxAttempts = 60) {
  for (let i = 0; i < maxAttempts; i++) {
    const payment = await getPayment(paymentId);
    
    if (payment.status === 'completed') {
      return payment;
    }
    
    if (payment.status === 'expired' || payment.status === 'cancelled') {
      throw new Error(`Pagamento ${payment.status}`);
    }
    
    // Aguardar 5 segundos
    await new Promise(resolve => setTimeout(resolve, 5000));
  }
  
  throw new Error('Timeout aguardando pagamento');
}
```

### Opção B: Webhooks (Recomendado)

Configure um webhook para receber notificações:

```javascript
app.post('/webhook', express.json(), async (req, res) => {
  const event = req.body;
  
  if (event.type === 'payment.completed') {
    const payment = event.data;
    
    // Processar pagamento
    await processPayment(payment.id);
    
    // Notificar cliente
    await notifyCustomer(payment.customer_email);
  }
  
  res.json({ success: true });
});
```

## Passo 4: Processar Pagamento

Após receber a confirmação, processe o pagamento:

```javascript
async function processPayment(paymentId) {
  // Buscar pagamento
  const payment = await getPayment(paymentId);
  
  if (payment.status !== 'completed') {
    throw new Error('Pagamento não foi confirmado');
  }
  
  // Atualizar banco de dados
  await db.orders.update(payment.metadata.order_id, {
    status: 'paid',
    payment_id: payment.id,
    paid_at: new Date(payment.paid_at)
  });
  
  // Enviar email de confirmação
  await sendEmail(payment.customer_email, {
    subject: 'Pagamento Confirmado',
    template: 'payment_confirmed',
    data: {
      amount: payment.amount,
      description: payment.description
    }
  });
  
  // Processar pedido
  await fulfillOrder(payment.metadata.order_id);
}
```

## Exemplo Completo

### Frontend (React)

```jsx
import { useState } from 'react';

export function CheckoutPage() {
  const [paymentId, setPaymentId] = useState(null);
  const [loading, setLoading] = useState(false);

  async function handleCheckout(amount, description) {
    setLoading(true);
    
    try {
      const response = await fetch('/api/create-payment', {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify({ amount, description })
      });
      
      const data = await response.json();
      setPaymentId(data.payment_id);
    } finally {
      setLoading(false);
    }
  }

  if (paymentId) {
    return <PixPayment paymentId={paymentId} />;
  }

  return (
    <button 
      onClick={() => handleCheckout(150.50, 'Compra de produto')}
      disabled={loading}
    >
      {loading ? 'Processando...' : 'Pagar com PIX'}
    </button>
  );
}
```

### Backend (Express)

```javascript
app.post('/api/create-payment', async (req, res) => {
  const { amount, description } = req.body;
  
  try {
    const response = await fetch('https://api.zynxwallet.com.br/v1/payments/create', {
      method: 'POST',
      headers: {
        'X-API-Key': process.env.ZYNX_API_KEY,
        'Content-Type': 'application/json'
      },
      body: JSON.stringify({
        amount,
        description,
        customer_email: req.user.email
      })
    });
    
    const data = await response.json();
    
    res.json({
      payment_id: data.data.id,
      qr_code_url: data.data.qr_code_url
    });
  } catch (error) {
    res.status(500).json({ error: error.message });
  }
});
```

## Boas Práticas

<Tip>
  **Dicas para integração PIX:**
  - Use webhooks em vez de polling
  - Valide sempre o status do pagamento
  - Armazene o ID do pagamento
  - Implemente tratamento de erros
  - Teste com valores pequenos primeiro
  - Use metadados para rastrear informações
</Tip>
