# Cancelar Pagamento

Cancele um pagamento que ainda não foi pago.

## Endpoint

```
POST /v1/payments/{payment_id}/cancel
```

## Autenticação

Requer autenticação via API Key ou JWT Token.

## Parâmetros

| Parâmetro | Tipo | Localização | Descrição |
|-----------|------|------------|-----------|
| `payment_id` | string | URL | ID do pagamento |
| `reason` | string | Body | Motivo do cancelamento (opcional) |

## Exemplo de Requisição

### cURL

```bash
curl -X POST https://api.zynxwallet.com.br/v1/payments/pay_abc123def456/cancel \
  -H "X-API-Key: sua_api_key_aqui" \
  -H "Content-Type: application/json" \
  -d '{
    "reason": "Cliente solicitou cancelamento"
  }'
```

### JavaScript

```javascript
async function cancelPayment(paymentId, reason = '') {
  const response = await fetch(
    `https://api.zynxwallet.com.br/v1/payments/${paymentId}/cancel`,
    {
      method: 'POST',
      headers: {
        'X-API-Key': process.env.ZYNX_API_KEY,
        'Content-Type': 'application/json'
      },
      body: JSON.stringify({ reason })
    }
  );

  const data = await response.json();
  
  if (!data.success) {
    throw new Error(data.message);
  }

  return data.data;
}

// Usar
const cancelled = await cancelPayment(
  'pay_abc123def456',
  'Cliente solicitou cancelamento'
);
console.log('Pagamento cancelado:', cancelled.id);
```

### Python

```python
import requests

def cancel_payment(payment_id, reason=''):
    response = requests.post(
        f'https://api.zynxwallet.com.br/v1/payments/{payment_id}/cancel',
        headers={
            'X-API-Key': os.getenv('ZYNX_API_KEY'),
            'Content-Type': 'application/json'
        },
        json={'reason': reason}
    )

    data = response.json()
    
    if not data['success']:
        raise Exception(data['message'])

    return data['data']

# Usar
cancelled = cancel_payment(
    'pay_abc123def456',
    'Cliente solicitou cancelamento'
)
print(f"Pagamento cancelado: {cancelled['id']}")
```

## Códigos de Erro

### 400 - Bad Request

```json
{
  "success": false,
  "error": "invalid_status",
  "message": "Não é possível cancelar um pagamento que já foi pago"
}
```

**Causas comuns:**
- Pagamento já foi confirmado
- Pagamento já foi cancelado
- Pagamento expirou

### 404 - Not Found

```json
{
  "success": false,
  "error": "payment_not_found",
  "message": "Pagamento não encontrado"
}
```

## Regras de Cancelamento

<Warning>
  **Importante:** Você só pode cancelar pagamentos com status `pending`.
</Warning>

| Status | Pode Cancelar? |
|--------|----------------|
| `pending` | ✅ Sim |
| `completed` | ❌ Não |
| `cancelled` | ❌ Não (já cancelado) |
| `expired` | ❌ Não |
| `failed` | ❌ Não |
