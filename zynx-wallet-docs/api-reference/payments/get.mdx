# Obter Pagamento

Obtenha os detalhes completos de um pagamento específico.

## Endpoint

```
GET /v1/payments/{payment_id}
```

## Autenticação

Requer autenticação via API Key ou JWT Token.

## Parâmetros

| Parâmetro | Tipo | Localização | Descrição |
|-----------|------|------------|-----------|
| `payment_id` | string | URL | ID do pagamento |

## Exemplo de Requisição

### cURL

```bash
curl -X GET https://api.zynxwallet.com.br/v1/payments/pay_abc123def456 \
  -H "X-API-Key: sua_api_key_aqui"
```

### JavaScript

```javascript
async function getPayment(paymentId) {
  const response = await fetch(
    `https://api.zynxwallet.com.br/v1/payments/${paymentId}`,
    {
      headers: {
        'X-API-Key': process.env.ZYNX_API_KEY
      }
    }
  );

  const data = await response.json();
  
  if (!data.success) {
    throw new Error(data.message);
  }

  return data.data;
}

// Usar
const payment = await getPayment('pay_abc123def456');
console.log('Status:', payment.status);
console.log('Valor:', payment.amount);
```

### Python

```python
import requests

def get_payment(payment_id):
    response = requests.get(
        f'https://api.zynxwallet.com.br/v1/payments/{payment_id}',
        headers={
            'X-API-Key': os.getenv('ZYNX_API_KEY')
        }
    )

    data = response.json()
    
    if not data['success']:
        raise Exception(data['message'])

    return data['data']

# Usar
payment = get_payment('pay_abc123def456')
print(f"Status: {payment['status']}")
print(f"Valor: {payment['amount']}")
```

## Possíveis Status

| Status | Descrição |
|--------|-----------|
| `pending` | Pagamento aguardando confirmação |
| `completed` | Pagamento confirmado e recebido |
| `cancelled` | Pagamento foi cancelado |
| `expired` | Pagamento expirou |
| `failed` | Pagamento falhou |

## Códigos de Erro

### 404 - Not Found

```json
{
  "success": false,
  "error": "payment_not_found",
  "message": "Pagamento não encontrado"
}
```

**Solução:** Verifique se o ID do pagamento está correto.

### 401 - Unauthorized

```json
{
  "success": false,
  "error": "invalid_api_key",
  "message": "API Key inválida ou expirada"
}
```

## Polling para Confirmação

Para aguardar a confirmação de um pagamento, você pode fazer polling:

### JavaScript com Retry

```javascript
async function waitForPaymentCompletion(paymentId, maxAttempts = 60) {
  let attempts = 0;
  
  while (attempts < maxAttempts) {
    const payment = await getPayment(paymentId);
    
    if (payment.status === 'completed') {
      return payment;
    }
    
    if (payment.status === 'expired' || payment.status === 'cancelled') {
      throw new Error(`Pagamento ${payment.status}`);
    }
    
    // Aguardar 5 segundos antes de tentar novamente
    await new Promise(resolve => setTimeout(resolve, 5000));
    attempts++;
  }
  
  throw new Error('Timeout aguardando confirmação do pagamento');
}

// Usar
try {
  const payment = await waitForPaymentCompletion('pay_abc123def456');
  console.log('Pagamento confirmado!', payment);
} catch (error) {
  console.error('Erro:', error.message);
}
```

## Alternativa: Usar Webhooks

<Tip>
  **Recomendação:** Use webhooks em vez de polling para melhor performance e menor latência.
</Tip>

```javascript
// Ao invés de fazer polling:
const payment = await waitForPaymentCompletion(paymentId);

// Configure um webhook:
app.post('/webhook/payment', (req, res) => {
  const event = req.body;
  
  if (event.type === 'payment.completed') {
    console.log('Pagamento confirmado!', event.data);
    // Processar pagamento
  }
  
  res.json({ success: true });
});
```
