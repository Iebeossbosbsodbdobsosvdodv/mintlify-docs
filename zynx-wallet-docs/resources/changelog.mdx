# Changelog

Acompanhe todas as atualizações, melhorias e correções da Zynx Wallet.

## v1.5.0 - 10 de Fevereiro de 2024

### ✨ Novas Funcionalidades

- **Transferências P2P**: Novos endpoints para transferências entre usuários Zynx Wallet
- **Webhooks Melhorados**: Suporte a mais eventos e retry automático aprimorado
- **Dashboard Redesenhado**: Interface mais intuitiva e responsiva
- **Relatórios**: Novos relatórios de transações e análises

### 🔧 Melhorias

- Performance de API aumentada em 40%
- Melhor tratamento de erros com mensagens mais claras
- Documentação expandida com mais exemplos
- Suporte a mais bancos para saques

### 🐛 Correções

- Corrigido problema com QR Code em alguns navegadores
- Corrigido erro ao cancelar pagamentos expirados
- Corrigido problema com webhooks duplicados

## v1.4.0 - 15 de Janeiro de 2024

### ✨ Novas Funcionalidades

- **Autenticação JWT**: Novo sistema de tokens JWT para maior segurança
- **Rate Limiting Aprimorado**: Limites mais generosos para usuários premium
- **Metadata Customizada**: Suporte a dados customizados em pagamentos

### 🔧 Melhorias

- Melhor validação de entrada
- Melhor tratamento de erros
- Documentação melhorada

### 🐛 Correções

- Corrigido problema com API Key expirada
- Corrigido erro ao listar pagamentos com filtros

## v1.3.0 - 20 de Dezembro de 2023

### ✨ Novas Funcionalidades

- **Saques Automáticos**: Agendar saques automáticos
- **Notificações por Email**: Notificações de pagamentos e saques
- **Histórico de Transações**: Visualizar histórico completo

### 🔧 Melhorias

- Interface de dashboard mais limpa
- Melhor performance de carregamento
- Melhor suporte a mobile

### 🐛 Correções

- Corrigido problema com saldo incorreto
- Corrigido erro ao criar pagamento com valor grande

## v1.2.0 - 25 de Novembro de 2023

### ✨ Novas Funcionalidades

- **Webhooks**: Sistema de notificações em tempo real
- **Múltiplas Contas Bancárias**: Suporte a múltiplas contas para saques
- **Filtros Avançados**: Filtros mais poderosos para listar pagamentos

### 🔧 Melhorias

- Melhor segurança de API Key
- Melhor documentação
- Melhor suporte ao cliente

### 🐛 Correções

- Corrigido problema com pagamentos pendentes
- Corrigido erro ao atualizar perfil

## v1.1.0 - 30 de Outubro de 2023

### ✨ Novas Funcionalidades

- **Saques**: Sistema de saques para conta bancária
- **Planos**: Diferentes planos com diferentes limites
- **Dashboard**: Painel de controle completo

### 🔧 Melhorias

- Melhor interface de usuário
- Melhor performance
- Melhor documentação

### 🐛 Correções

- Corrigido problema com QR Code
- Corrigido erro ao criar pagamento

## v1.0.0 - 15 de Setembro de 2023

### ✨ Funcionalidades Iniciais

- **Pagamentos PIX**: Criar e gerenciar pagamentos PIX
- **API RESTful**: API completa para integração
- **Autenticação**: Sistema de autenticação com API Keys
- **Documentação**: Documentação completa da API

## Plano de Desenvolvimento

### Próximas Versões

- **v1.6.0**: Integração com mais gateways de pagamento
- **v1.7.0**: Sistema de facturação automática
- **v1.8.0**: Suporte a criptomoedas
- **v2.0.0**: Redesign completo da plataforma

## Política de Suporte

- **Versão Atual**: v1.5.0 (suporte completo)
- **Versões Anteriores**: Suporte por 12 meses após lançamento de nova versão
- **Versões Descontinuadas**: Sem suporte

## Como Atualizar

A Zynx Wallet é atualizada automaticamente. Você não precisa fazer nada. Todas as mudanças são retrocompatíveis.

## Reportar Problemas

1. Verifique o [FAQ](/resources/faq)
2. Verifique a [Documentação](/introduction)
