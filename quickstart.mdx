## 1. Obtenha sua Chave de API

Para começar, você precisará de uma chave de API válida. Você pode gerá-la no seu painel de controle.

<Note>
  Todas as chaves de API da Zynx começam com o prefixo `zynx_`. Nunca compartilhe sua chave secreta.
</Note>

## 2. Base URL

Todas as requisições devem ser feitas para a seguinte URL base:

```text
https://api.zynxwallet.com.br/api/v1
```

## 3. Autenticação

A autenticação é feita através do cabeçalho `X-API-Key` em todas as requisições.

```bash
X-API-Key: zynx_sua_chave_aqui
```

## 4. Sua Primeira Requisição

Tente criar um pagamento de teste usando o comando `curl` abaixo:

```bash
curl --request POST \
  --url https://api.zynxwallet.com.br/api/v1/payment/create \
  --header 'Content-Type: application/json' \
  --header 'X-API-Key: zynx_sua_chave' \
  --data '{
  "value": 5.00,
  "description": "Meu primeiro pagamento",
  "coverFee": true
}'
```
